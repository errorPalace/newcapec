升级日志：
版本： 1.0.0
内容：
	*1.0版本。 H5门户后台

升级注意事项：
=查JDK版本,必须1.6+  tomcat版本7.0+或者jetty8.0+ (支持servlet3.0的容器皆可)

升级文件名：H5校园卡门户.1.0.0升级包.zip

升级类型：全量升级方式(war)

文件清单：
	1、campus-h5-service-1.0.0-release.war
	2、h5-setup-db.sql
	3、readme.txt

升级步骤：
1、执行数据库脚本：   创建数据库名称为：CAMPUS_QUICKACCESS,执行数据库脚本：h5-setup-db.sql
2、升级项目：         解压campus-h5-service-1.0.0-release.war为ROOT文件夹，然后放入TOMCAT\webapp\ROOT目录（war可修改为zip后解压,ROOT文件夹如存在请备份后替换）
3、部署到TOMCAT：    上如步骤完完成后，启动项目 (该版本修为全量更新,替换ROOT文件夹)
4、检查启动是否正常： 检查是有有异常,访问管理后台页面是否能正常打开，后台功能是否正常使用
